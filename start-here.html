01Required Viewing

Before you do anything, watch both of these documentaries in full. These are the benchmarks for density, pacing, and visual style. Every decision you make should be measured against these videos.

JP Morgan Documentary (12:31)

Study: overall pacing, B-roll density, custom visuals (timelines, data charts), section transitions

David Solomon Documentary (18:22)

Study: image sourcing, highlight overlays on articles, metaphor clips, era-appropriate visuals

While watching, pay attention to how rarely the host is on screen without a visual cut. Count the seconds between cuts. Notice how every single sentence has something visual happening. That is the standard.

Non-Negotiable Rule

The host is on screen only ~15% of the time. ~85% of the video must be B-roll, still images, or custom visuals. Most host streaks are 2-5 seconds. Stretches of 10-15 seconds happen during key narrative moments. Hard ceiling: ~15 seconds — never exceed this without a visual cut.
