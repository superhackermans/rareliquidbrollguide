Annotation Format

Exact formats for B-roll, images, audio, and overlay annotations.

14B-Roll Annotation Format

Every B-roll reference must follow this exact format. No exceptions.

Correct[https://www.youtube.com/watch?v=QZ85VPgCOCM, 0:18 - 0:29] [https://www.youtube.com/watch?v=nbs_s6_L_ao, 22:10 - 22:14]
Wrong — Do Not Do These[Goldman Sachs CEO Interview, 0:18 - 0:29] ← No titles, raw URLs only [https://youtube.com/watch?v=abc, around 3 min] ← No vague timestamps [https://youtube.com/watch?v=abc] ← Never omit timestamps

Alternate Options

[https://youtube.com/watch?v=abc, 4:05 - 4:09] OR [https://youtube.com/watch?v=xyz, 16:10 - 16:42]
