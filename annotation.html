Annotation Format

Exact formats for B-roll, images, audio, and overlay annotations.

13B-Roll Annotation Format

When you find a video clip to use as B-roll, you annotate it directly in the script next to the line it belongs to. The format is the raw YouTube URL plus the exact start and end timestamps in brackets. No video titles, no vague time references.

Correct Format

Always use this[https://www.youtube.com/watch?v=QZ85VPgCOCM, 0:18 - 0:29] [https://www.youtube.com/watch?v=nbs_s6_L_ao, 22:10 - 22:14]

Common Mistakes

These will get flagged and sent back for revision:

Wrong — Do Not Do These[Goldman Sachs CEO Interview, 0:18 - 0:29] ← No titles, raw URLs only [https://youtube.com/watch?v=abc, around 3 min] ← No vague timestamps [https://youtube.com/watch?v=abc] ← Never omit timestamps

Providing Alternate Options

If you find two good clips for the same script line, include both separated by "OR" so the best one can be chosen:

[https://youtube.com/watch?v=abc, 4:05 - 4:09] OR [https://youtube.com/watch?v=xyz, 16:10 - 16:42]

14Image Annotation Format

Still images are numbered sequentially (1.png, 2.png, 3.png) and uploaded as a folder to Slack. In the script, you reference them by number. You can also add instructions for how the image should be displayed on screen — highlighting a phrase, zooming in, or panning across it.

15Audio & Overlay Formats

Beyond standard B-roll and images, there are three special annotation types: clips where the original audio plays, on-screen graphic overlays, and post-rough-cut timestamp anchoring.

B-Roll with Audio

When a clip's original audio should play instead of Ben's voiceover, add WITH AUDIO at the end. This is rare — only 3-5 times per video.

[https://www.youtube.com/watch?v=xyz, 4:05 - 4:09, WITH AUDIO]

Overlays

Overlays are on-screen graphics that appear over the footage — logos, arrows, labels, or text. Describe exactly what should appear and when:

[show Goldman Sachs logo when host says "Goldman"] [arrow pointing to player A labeled "Solomon", player B labeled "Goldman Sachs"] [on-screen text: "95 hours per week" when host says the statistic]

Timestamp Anchoring (Post Rough-Cut)

After the rough-cut is done, go back and add the video timeline position (MMSS format) so each asset is placed at the exact moment in the video:

FormatMMSS - [asset reference] 0342 - [https://youtube.com/watch?v=abc, 1:03 - 1:08]
