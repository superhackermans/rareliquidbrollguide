B-Roll, Still Images & Custom Visuals Guidelines

The single source of truth for anyone preparing visuals for a rareliquid documentary. Follow these guidelines exactly to eliminate revision rounds.

75-90 B-roll clips55-70 still images1-3 custom visuals0 gaps allowed

01Required Viewing

Before you do anything, watch both of these documentaries in full. These are the benchmarks for density, pacing, and visual style. Every decision you make should be measured against these videos.

JP Morgan Documentary (12:31)

Study: overall pacing, B-roll density, custom visuals (timelines, data charts), section transitions

David Solomon Documentary (18:22)

Study: image sourcing, highlight overlays on articles, metaphor clips, era-appropriate visuals

While watching, pay attention to how rarely the host is on screen without a visual cut. Count the seconds between cuts. Notice how every single sentence has something visual happening. That is the standard.

Non-Negotiable Rule

The host is on screen only ~13% of the time. ~87% of the video must be B-roll, still images, or custom visuals. Most host streaks are 2-5 seconds. Stretches of 10-15 seconds happen during key narrative moments. Hard ceiling: ~15 seconds — never exceed this without a visual cut.

02The Problem We Solve

In past productions, the initial B-roll pass covered only about 50-60% of what was needed. The most common issues were:

  • Not enough total clips
  • Clips loosely related but not specific to the sentence being spoken
  • Generic stock footage used when real interview/news clips existed
  • Present-day footage used for historical events (e.g., showing 2024 Solomon during a 1986 Drexel section)
  • Boring or plain clips chosen when more visually interesting alternatives existed

04The No-Gap Rule

Every single line of the script must have at least one visual option tagged to it. If a line discusses a person, there should be footage or an image of that person. If a line discusses a company, there should be a logo, office footage, or news clip. If a line discusses a concept, there should be a relevant chart, document, or illustrative visual.

There should be zero orphan lines with no visual attached. If you cannot find anything, flag it explicitly rather than leaving it blank:

[NO B-ROLL FOUND - need alternative]

05Visual Variety Rule

Never have 3 or more consecutive still images without a video clip in between. If you find yourself stacking images back-to-back, search harder for a video source to break up the sequence. The visual rhythm should alternate between video clips, still images, and the host.

Approximately 16 visual changes happen per minute in our documentaries. That means a new cut roughly every 3-4 seconds.

06Asset Reuse Policy

Some reuse is normal. In the Solomon documentary:

  • 23 still image sources are reused across 96 placements
  • 11 B-roll sources are reused across 96 placements
  • A photo of the subject appears multiple times when discussing different career phases

The key rule: always prioritize UNIQUE sources to hit the minimums (75-90 B-roll, 55-70 images) first. Reuse is acceptable only after you have exhausted unique options. Do not start with a plan to reuse heavily.

07B-Roll Video

Video footage that plays over the host's voiceover. The viewer sees the footage but hears Ben narrating. This is the primary visual type, making up ~59% of screen time.

When to Use B-Roll

  • Showing a person in action or speaking (interview footage, news coverage, conference keynotes)
  • Demonstrating a location, building, or environment
  • Illustrating a concept, process, or activity
  • Filling gaps between still images to maintain visual variety and pacing
  • The original audio from the clip is not relevant to the narrative (most use cases)

Rules

  • 3-10 seconds per clip (ideal: 4-7 seconds). No isolated clips under 3 seconds.
  • Must match the era discussed in the script — no 2024 footage for 1986 events
  • Must pass the relevance test: if watched on mute, could you guess the topic?
  • Always pick the more interesting clip when two exist
  • Never reuse the same clip for two different script lines (unless revisiting a topic)
  • Stock footage is a last resort (under 10% of total)

How to Annotate

Format[https://www.youtube.com/watch?v=QZ85VPgCOCM, 0:18 - 0:29]
Visual Distinction

When an editor sees B-roll with no "WITH AUDIO" marker, they assume the host voiceover plays over it. The video footage is background; the audio is foreground.

08Still Images

A static image displayed on screen, shown for 2-6 seconds. The editor applies zoom/pan (Ken Burns effect) to 91-93% of them — that's the default. Still images make up ~22% of screen time.

Image Types (In Order of Preference)

  1. News article screenshots with key phrases highlighted — the signature rareliquid look. Always find the original article, not a tweet about it.
  2. Official documents and filings — SEC filings, court docs, press releases, earnings reports. Crop to the relevant section.
  3. Data charts and graphs — from reputable sources (Bloomberg, WSJ, company filings). Must be legible at 1080p.
  4. High-resolution photographs — professional headshots, event photos, campus/building shots. For historical subjects, search "[name] young" for age-appropriate photos.
  5. Social media screenshots — only when directly relevant and from notable accounts.

Quality Requirements

  • Minimum resolution: 1280x720 (ideally 1920x1080 or higher)
  • No watermarks, no compression artifacts
  • Crop to relevant content — no browser chrome, tabs, or bookmarks
  • If a highlight is needed, note it: [pic 7, highlight "specific phrase"]

Naming Convention

Name every image sequentially: 1.png, 2.png, 3.png. Upload the entire folder to Slack as a batch.

How to Annotate

Formats[pic 1] [pic 7, highlight "specific text to underline"] [pic 12, zoom] [pic 15, pan left-to-right] [pic 8, zoom and highlight "key phrase"]

09B-Roll with Audio

Same as B-roll, but the original audio plays instead of Ben's voiceover. Very rare: ~2% of total video, only 3-5 uses per 15-minute video.

When to Use (Sparingly)

  • A CEO or public figure makes a critical statement that is more powerful heard in their own voice
  • A news anchor is breaking a major story and their delivery adds drama and credibility
  • An interview subject makes a quote that deserves to be heard directly, not summarized
  • A moment of genuine emotion or emphasis that loses impact if paraphrased
Warning

Overusing "WITH AUDIO" dilutes its impact. If every other clip has original audio, none of them feel special. Reserve this for moments that give the viewer chills. Default to standard B-roll with voiceover.

How to Annotate

Format[https://www.youtube.com/watch?v=xyz, 4:05 - 4:09, WITH AUDIO]

10Custom Visuals (HTML)

Custom-built HTML files rendered as graphics. Only 1-3 per entire video. Do not overuse.

When to Flag for a Custom Visual

  • Career timelines (e.g., Irving Trust → Drexel → Bear Stearns → Goldman)
  • Financial data: revenue, profit margins, stock price comparisons
  • Comparison tables: two candidates side by side, company metrics vs. peers
  • Company splits, mergers, organizational changes
  • Key statistics or quotes that deserve visual emphasis

How to Annotate

Format[custom visual: Solomon career timeline Irving → Drexel → Bear → Goldman] [custom visual: Goldman stock vs JPM vs MS since 2018]

Quality Checklist

  • All text legible at 1080p
  • Data sourced and cited
  • Self-contained HTML file (no external dependencies)
  • 16:9 aspect ratio

Critical

14Historical Accuracy

This is where past deliveries failed the most. If the script is talking about something that happened in 1990, you cannot use footage of that person from 2024.

  • If the script discusses events from a specific era, the B-roll must be from that era or visually neutral
  • When a person is discussed at a young age, search for "[person name] young" or "[person name] early career"
  • If no era-appropriate footage exists, use a still image (news clipping, document, yearbook photo) rather than modern footage
  • For historical companies (Drexel Burnham Lambert, Bear Stearns), look for archival news coverage, not modern retrospectives
Real Example

In the Solomon documentary, a clip at timestamp 4:20 had to be removed because it showed Solomon in the present day during a section about his time at Drexel in the 1980s. A clip of modern computers was also replaced because it showed modern technology during a 1980s section.

15Relevance Standard & the Better Option Rule

Every clip must pass this test: if someone watched the clip on mute without context, could they reasonably guess what the narrator is talking about? If no, the clip is not relevant enough.

Bad

Script says "Solomon joins Drexel Burnham Lambert" and B-roll shows a generic modern trading floor.

Good

Script says "Solomon joins Drexel Burnham Lambert" and B-roll shows a 1980s news clip about Drexel, or an archival photo of Drexel's office, or footage of Michael Milken.

The "Better Option" Rule: If you find two clips for the same moment, always use the more visually interesting one. If a still image is more relevant and interesting than a generic video clip, use the image.

19Hook Section (First 60-90 Seconds)

The hook is the most visually dense part of the video. The goal is to grab attention immediately.

  • Almost zero host face time in the first 10-15 seconds
  • Rapid-fire clips (2-4 seconds each) from multiple sources before the host appears
  • Each hook question paired with a specific, relevant visual — not a generic Wall Street shot
  • In the Solomon doc, the hook cuts between an interview clip, a Bloomberg segment, DJ footage at Lollapalooza, and a news reel — 4 different visual sources in the first 45 seconds
  • The hook should feel like a movie trailer: fast, punchy, every frame earns its place

20Section Transitions

When the script moves between major sections (e.g., "Early Career" to "Goldman Era"), there should be a visual bridge. Do not just cut from the host talking about one era to another.

  • Use a "bridge clip" from an interview or speech that introduces the next topic (e.g., Blankfein saying "I want to introduce the next CEO of Goldman Sachs")
  • Custom timeline visuals or historical footage to reset the viewer's mental timeline
  • If no bridge clip exists, use a full-screen title card, a dramatic still image, or a custom visual

21Metaphor & Analogy Clips

When the script uses a metaphor, the B-roll should visualize that metaphor literally. This is one of the most impactful techniques in our documentaries.

Example from Solomon Doc

When the script says Goldman rejected Solomon, the B-roll shows a basketball player getting his shot blocked. Arrows label the dunker "Solomon" and the blocker "Goldman Sachs." This is the kind of creative, specific visual that makes our videos distinctive.

When you encounter a metaphor in the script, think: what would this look like if I filmed it? Then find a clip that matches.

22Article & Document Highlights

One of the most distinctive rareliquid techniques: showing news articles on screen with specific phrases highlighted.

  • When the script references a specific claim, show the actual article with the relevant phrase highlighted
  • When the script references a data point (e.g., "95 hours a week"), show the source document with that number highlighted
  • Always find the original article, not a summary or a tweet about it
  • Screenshot the article itself, crop to the relevant section, and note which phrase to highlight

23Show, Don't Tell

Every time the script names a person, company, place, or product, something visual should appear.

Script Mentions...You Should Show...
A person by nameFootage of that person (interview, news clip) or a high-quality photo
A company by nameCompany logo overlay, office building, or news coverage of that company
A specific locationPhoto or footage of that location (campus, building, city)
A product or serviceProduct screenshot, app interface, or promotional material
A financial figure or metricA chart, filing screenshot, or custom HTML visual with the number
A quote or statementThe source article/document with the quote highlighted
A historical eventEra-appropriate news footage or archival photography
A concept (e.g., "junk bonds")An explainer visual, relevant news clip, or custom graphic

Ad sections should be treated with the same visual density as main content. Product screenshots, course interfaces, logos — keep the screen active.

24The 7-Step Workflow

This is the exact workflow for every script. Do not skip steps.

Step 1: Watch the Reference Documentaries

Watch both the JPM doc and the Solomon doc in full. Note the visual rhythm.

Step 2: Full Script Read-Through

Read the entire script once without annotating. Understand the arc, key people, companies, events, and historical time periods covered.

Step 3: Entity Extraction

List every unique entity:

  • People (full names, roles, time periods discussed)
  • Companies (including former names, subsidiaries, and the era being discussed)
  • Events (IPOs, scandals, promotions, product launches, with specific years)
  • Data points (revenue figures, percentages, timelines, survey results)
  • Quotes and statements (attributed lines that might need on-screen text or article source)
  • Metaphors and analogies ("cage match," "caught the bus by the tailpipe" — these need creative visual treatment)

Step 4: Bulk Search Phase

For each entity, conduct a dedicated search session. Batch searches by entity, not line-by-line. Log every usable source in a spreadsheet with: URL, timestamp range, description, era/year, and which script lines it serves.

Step 5: Line-by-Line Assignment

Assign at least one visual to every line using the "Show, Don't Tell" table. Zero orphan lines.

Step 6: Gap Audit

Check for:

  • Any line with no visual assigned → go back to Step 4
  • Host face on screen for more than 5 seconds → add B-roll or images
  • 3+ consecutive still images → find a video clip to break it up
  • Stock footage used → try one more time for a Tier 1-3 alternative
  • Era mismatches → replace with era-appropriate alternative
  • Boring clips → search for a more visually interesting option
  • Section transitions → ensure each has a visual bridge

Step 7: Final Count Check

Before submitting, verify:

  • Total B-roll clips: meets per-minute minimum
  • Total still images: meets per-minute minimum
  • Stock footage: under 10%
  • Lines with no visual: zero
  • Era mismatches: zero

If any check fails, go back to Step 4 before submitting.
