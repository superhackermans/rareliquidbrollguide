# rareliquid B-Roll Production Guidelines

Internal reference site for the rareliquid video production pipeline. These guidelines are used by the B-roll finder to ensure consistent, high-quality visual sourcing for our documentary-style YouTube videos.

**Live site:** https://superhackermans.github.io/rareliquidbrollguide/

## Structure

| File | Content |
|---|---|
| `index.html` | Landing page |
| `start-here.html` | Required viewing, per-minute asset calculator |
| `asset-types.html` | B-Roll, Still Images, B-Roll with Audio, Custom Visuals, annotated example |
| `sourcing.html` | Source priority, clip duration rules, historical accuracy, relevance standard |
| `techniques.html` | Hook section, transitions, metaphors, article highlights, show don't tell |
| `reference.html` | Common mistakes, deliverables checklist, quick reference card |
| `css/style.css` | Shared styles (dark/light theme, sidebar, calculator, cards) |
| `js/main.js` | Theme toggle, sidebar navigation, asset calculator |

## Tech

- Static HTML/CSS/JS — no build step, no dependencies
- Hosted on GitHub Pages
- Dark mode by default, light mode toggle (persisted via localStorage)
- Fonts: Instrument Serif, DM Sans, JetBrains Mono (Google Fonts)

## Internal Use Only

This document is not meant to be shared outside of the organization.
